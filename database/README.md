# قاعدة بيانات ARABYA.NET على GitHub

هذا الملف يمثل نسخة قاعدة البيانات المركزية داخل المستودع.

الملف الأساسي:

`database/arabya-db.json`

يتم تحديثه من خلال Google Apps Script فقط، لأن وضع مفتاح GitHub داخل ملفات الموقع أو JavaScript العام سيكون خطراً جداً. المتصفح يرسل البيانات إلى رابط Google Apps Script، والسكريبت يحفظها في:

- Google Sheets للمراجعة السريعة.
- ملف JSON داخل GitHub للنسخة المركزية.
- التخزين المحلي داخل المتصفح كنسخة مؤقتة وسريعة.

محتوى قاعدة البيانات:

- `teachers`: حسابات المعلمين.
- `students`: حسابات الطلاب وأكواد الاشتراك.
- `exams`: الامتحانات والأسئلة.
- `results`: نتائج الطلاب وإجاباتهم وتعديلات المعلم.
- `auditLog`: سجل مختصر للتعديلات والمزامنة.

تنبيه أمني: لا تضع GitHub token في ملفات الموقع مطلقاً.
