# ARABYA.NET — جاهزية الإطلاق

## التحقق السريع

```bash
node --check app.js
node --test tests/*.test.js
```

## قبل الإطلاق (المعلم)

1. غيّر كلمة مرور المعلم الافتراضية `TEACHER2026` فور أول دخول.
2. في تبويب **الربط بـ Google Sheets**: الصق كود Apps Script → انشر Web App → **Anyone** → احفظ رابط `/exec`.
3. أنشئ امتحاناً واضغط **نسخ الرابط** — يجب أن يحتوي على `exam=` و `teacher=` و `s=`.
4. اضغط **اختبار المزامنة** على بطاقة الامتحان.

## للطلاب

- الرابط المباشر يحمّل إعدادات المزامنة تلقائياً (`s=`).
- الاسم إلزامي؛ ID وكود الاشتراك اختياريان (الكود 5 أرقام، `00000` مشترك).
- بعد التصحيح اليدوي: زر **مزامنة الدرجات مع Google Sheets**.
