# تصدير البيانات بدون الكونسول (عند منع اللصق)

موقع ARABYA يمنع اللصق في الكونسول أحياناً بسبب حماية الامتحان. استخدم إحدى الطرق التالية.

---

## الطريقة 1 — الأسهل (من لوحة المعلم، بدون كونسول)

هذه الطريقة **كافية لنسخة احتياطية كاملة** للمعلمين والطلاب والامتحانات والنتائج وبنوك الأسئلة.

1. افتح **https://arabya.net**
2. سجّل دخول **المعلم** (أو السوبر أدمن)
3. من القائمة الجانبية اذهب إلى تبويب **الربط بـ Google Sheets** أو ابحث عن قسم:
   **«قاعدة البيانات الشاملة للمنصة (النسخ الاحتياطي والاستعادة)»**
4. اضغط الزر الأزرق:
   **«تحميل النسخة الاحتياطية الكاملة (JSON)»**
5. احفظ الملف باسم واضح، مثلاً:
   `arabya_full_export_2026-06-05.json`

هذا الملف يشمل تقريباً كل ما تحتاجه للعودة لهذه النسخة.

---

## الطريقة 2 — Bookmarklet (نقرة واحدة، بدون لصق في الكونسول)

1. افتح الملف التالي **على جهازك** (من المستودع أو بعد تنزيله):
   `backups/2026-06-05-pre-security-fixes/export-localstorage-bookmarklet.html`
2. اسحب الرابط **«تصدير ARABYA localStorage»** إلى شريط المفضلة (Bookmarks).
3. افتح **https://arabya.net** وسجّل دخول المعلم.
4. اضغط على الإشارة المرجعية من المفضلة **مرة واحدة**.
5. سيُنزّل ملف `arabya_localStorage_backup_YYYY-MM-DD.json` تلقائياً.

> لا تحتاج F12 ولا لصقاً في الكونسول.

---

## الطريقة 3 — Chrome Snippets (اللصق في محرر Snippets وليس Console)

في Chrome، اللصق **مسموح** في محرر Snippets وليس في تبويب Console.

1. افتح **https://arabya.net** وسجّل دخول المعلم.
2. اضغط **F12** → تبويب **Sources** (المصادر).
3. من الشريط الجانبي: **Snippets** → **+ New snippet**.
4. سمِّ المقتطف مثلاً: `arabya-export`
5. **هنا** الصق محتوى الملف `LOCALSTORAGE_EXPORT.js` (اللصق يعمل في هذا المحرر).
6. احفظ (Ctrl+S).
7. شغّل المقتطف: كليك يمين على اسم المقتطف → **Run**  
   أو حدده واضغط **Ctrl+Enter**.

---

## الطريقة 4 — Firefox (مقتطفات المطور)

1. افتح **https://arabya.net**
2. **F12** → **تصحيح الأخطاء** → **المقتطفات** (أو Debugger → Snippets)
3. **مقتطف جديد** → الصق `LOCALSTORAGE_EXPORT.js`
4. **تشغيل**

---

## الطريقة 5 — يدوياً من Application (للمفاتيح الأساسية فقط)

إذا فشلت كل الطرق، انسخ القيم الأربع الأهم يدوياً:

1. **F12** → **Application** (أو **التخزين** في Firefox)
2. **Local Storage** → `https://arabya.net`
3. انقر كل مفتاح وانسخ قيمته إلى ملف نصي:

| المفتاح | المحتوى |
|---------|---------|
| `arabya_teachers_db` | المعلمون |
| `arabya_students_db` | الطلاب |
| `arabya_exams_db` | الامتحانات |
| `arabya_results_db` | النتائج |
| `arabya_teacher_config` | إعدادات المزامنة + رابط Sheets |

احفظها في ملف JSON بهذا الشكل:

```json
{
  "arabya_teachers_db": [ ... ],
  "arabya_students_db": [ ... ],
  "arabya_exams_db": [ ... ],
  "arabya_results_db": [ ... ],
  "arabya_teacher_config": { ... }
}
```

---

## أيهما تختار؟

| الهدف | الطريقة |
|-------|---------|
| نسخة احتياطية قبل الإصلاحات | **الطريقة 1** (زر لوحة المعلم) |
| كل مفاتيح localStorage بما فيها التفضيلات | **الطريقة 2** (Bookmarklet) أو **3** (Snippets) |
| طوارئ فقط | **الطريقة 5** |

---

## استعادة لاحقاً

من لوحة المعلم → **استيراد قاعدة البيانات** → اختر ملف `arabya_full_export_*.json`.
